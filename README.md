# Vue CLI 安装

1. Node 版本要求：8.9 或更高版本 (推荐 8.11.0+)
1. 安装命令：npm install -g @vue/cli
1. 验证是否安装成功：vue --version


# 创建一个新项目
1. 命令：vue create hello-world
1. 回车进行安装


### Project setup
```
npm install
```
